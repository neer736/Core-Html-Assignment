Full Stack Web Development Tech Stack

HTML
HTML stands for Hyper Text Markup Language. It is the markup language for creating the structure and layout for wbpages. we write structure in html with the help of tags. with the help of HTML we can make tables,forms,webpages etcProperly structured HTML is crucial for web accessibility. It allows screen readers and other assistive technologies to interpret and present web content to users with disabilities.

CSS
CSS stands for Cascading Style Sheets.with the help of CSS, Developer can control the styling part of web pages. it allows developer to change color,font,spacing,positioning of HTML elements.so in short html can build layout or structure of web pages and css can make them beautiful and stylish.CSS allows for easy customization and theming of web applications. By changing CSS styles, developers can give a website a completely different look and feel.

JAVASCRIPT
Javascript is a programming language which is object Oriented, clint site scripting,interactive etc.JS is used to creat interective effects on web browser.with the help of js developer can make dynamic websites.js performs the logical opration on the code which is written in html and css basically it makes live to webpage. developer can control html and css both by Javascript.js can manipulate and validate data.JavaScript can also be used for server-side development. Developers can build entire web servers and backend APIs using JavaScript, making it a full-stack language.

Node.js
Node.js is the most popular backend framework for creating server side Javascript.It is an open source server side runtime environment that allows developers to run Javascript code on server. Node.js helps bridge the devide between server and clint.Node.js provides security features, it's essential for developers to follow best practices to secure their applications.

Express.js
Express.js is a back end application framework for creating APIs using Javascript.it can also provide server side functionality. Express.js usually used with Node.js to provide a server framework that enebles Node.js to perform a server side backend scripting language.

MongoDB
MongoDB is popular and open source NoSQL (Not only SQL)database managenent system, developers store database in it. Records in MongoDB database are called documents, and the field value includes numbers,strings,arrays,booleas or nested Documents. MongoDB is used for high volume data storage or we can say large sets of data.

React.js
React.js is popular and open source Javascript library for building UI(user interface) . it is created by Facebook.it is used for creating dynamic and interactive web applications. React allows developers to build reusable UI components. it is used to update user interface and response to changes in data.React uses JSX, an extension of JavaScript that allows developers to write HTML-like code within their JavaScript files.
